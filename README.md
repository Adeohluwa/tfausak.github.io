# [taylor.fausak.me][1]

Taylor Fausak's blog.

## Setup

You'll need [Ruby][2] and [Bundler][3].

``` sh
$ git clone https://github.com/tfausak/tfausak.github.io.git
$ cd tfausak.github.io
$ bundle install
$ bundle exec rake serve
# http://localhost:4000
```
